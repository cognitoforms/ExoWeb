---
title: Meta markup extension
description: A markup extension that binds to the display value of an adapter.
---

The `{#}` markup extension creates an adapter and then binds to its `displayValue`.
This allows you to easily leverage any custom formatting and parsing defined in the model.

Example:
```html
<input sys:value="{# SomeProperty}" />
```
